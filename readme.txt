== Description ==
There have a lot of options to make the styles awesome portfolio section. You will finsd a custom post type named Portfolio. Add some portfolio there and use the shortocde. There is the single portfolio template in where you can add the widgets as you wish. So install the new version.

Just use this <pre>['sgp-portfolio']</pre> shortcode anywhere you want.
